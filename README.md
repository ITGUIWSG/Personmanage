# Personmanage
人事管理系统
一.模块说明：
1.empinfocenter-----员工信息中心
2.orgmanage---------组织结构管理
3.personinfomanage--人事信息管理
4.reportmanage------报表管理
5.db----------------存放数据库连接类
6.system------------系统管理


二.结构说明：
1.dao------------存放JavaBean相对应的数据库执行操作
2.db-------------存放数据库驱动类
3.domain---------存放JavaBean
4.service--------存放servlet调用相关服务
5.servlet--------存放servlet


三.在WebRoot下的相应模块文件夹下建jsp
